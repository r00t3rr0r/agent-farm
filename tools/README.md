# Tools

Place local CLI tools here (n8n-cli, playwright, markdownlint, etc.).

These tools are invoked by the workflow steps and must be available in `PATH` or this directory.

## Required Tools (for full functionality)
- `python3` – Python 3.10+
- `git` – Version control
- `sqlite3` – State management
- `npx` – For Playwright/Cypress tests (optional)
- `n8n-cli` – For low-code workflow validation (optional)
- `markdownlint-cli` – For docs validation (optional)
